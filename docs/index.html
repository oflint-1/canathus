Features

Customisable

With custom validators, canathus form elements can be crafted to fit any type of input data.

Lightweight

With a module size of 2.7kB, adding canathus to your project adds minimal overhead, creating a slimline final package.

Simple

Canathus has a shallow learning curve, meaning it can be implemented into projects quickly and easily.

Open-Source

Being open-source, canathus is completely free to use in any personal or commercial projects!
